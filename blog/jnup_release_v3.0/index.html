JnUp经济学文献追踪工具v3.0更新

🥳JournalUpdater更新发布v3.0版本，此版本为重要更新版本，请及时更新。该版本更新日志如下：

修复与更新

  • 修改了界面UI，主要针对mac下的界面进行了适配，并添加了按钮的动态效果，移动、点击时颜色变化，并适配MacOS下的深色模式
  • 大幅度优化流程和代码
    • 登录界面右下角新增检查更新按钮，便于和三场五幕工具开发站 中发布的JnUp最新版比对，并提示跳转。

image

  • 登陆密码与MySQL数据进行比对，比对成功才可登陆。
  • 优化了按键逻辑，若切换期刊，则右侧表格清空且切换为不可用状态，避免不对应而导致出错。

image

  • 此外，只有当右侧表格出现时，下方导出目录中选择文件夹 才可启用，当导出目录中有地址时，导出 按钮才可启用。

image

  • 登录不再测试SCI-Hub线路，将原来检索 功能替换为SCI-Hub测速 ，取消检索 功能，SCI-Hub线路来自于MySQL数据库，分为官方源和互联网源，可手动勾选后一键测速，自动选择速度最快的线路，也可右键选择合适的线路，亦可自定义线路，选择完毕后确认退出。

image

  • 新增英文期刊右侧表格列，为标题的机器翻译，便于阅读。

image

  • 优化详情页统一性，没有校外登陆的期刊统一设置校外登陆 按钮不可用，无法下载的也设置PDF下载 不可用。

image

  • 一键翻译 仅在中文标题和中文摘要为空或读取出错时才可使用，大幅减少请求互联网的次数。
  • 支持对系统的判断，根据判断系统执行cmd语句（如ping ip）。
  • 全面支持MacOS，在Monterey 12.1版本上测试有效。
  • 在win和mac上均使用安装工具封装。

下一步计划

  • 将部分MySQL数据移植到本地用sqlite，减少非必要的访问。同时解决本地数据库增量更新问题。
  • 新增我的阅读列表功能，将待读文献加入列表中便于及时查看，该项更新需要本地数据库sqlite支持。
